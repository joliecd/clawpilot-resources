# Prompt: Update Single Milestone Date

**Best for:** Moving one MSX milestone to a new target date using Clawpilot browser control.

_Credit: Ryan Otis_

---

## Prompt

```
Please update milestone [MILESTONE ID or NAME] to a new target date of [DATE, e.g. June 15, 2026].
Here is the URL to the milestone: [FULL MSX URL]
```

---

## Notes

- Use the **full URL** from your browser address bar — Edge sometimes shows a shortened/vanity URL that Clawpilot can't resolve. Copy it fresh from the full address.
- Clawpilot may try a couple of different approaches to save the date before it succeeds — this is expected behavior
- Always verify the date change in MSX after Clawpilot confirms success
