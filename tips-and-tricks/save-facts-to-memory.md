# Tip: Save Facts to Memory

Clawpilot forgets everything between sessions. But if you find something you'll need again — a file location, SharePoint URL, key contact — you can tell it to remember permanently.

## How to save a memory

At the end of a session, just say:

> "Save [this/that] to your memory so I can reference it next time"

**Real example:** Finding the FY26 Account Alignment spreadsheet required navigating The A-Team Teams chat, finding the pinned tab, and resolving the SharePoint URL. One prompt saved all of that work — next session, just ask *"where is the FY26 Account Alignment file?"* and Clawpilot answers instantly.

## How to view your stored memories

In the Clawpilot app, click **Settings** (gear icon, bottom left) → **Memory**. You can view, edit, or delete any saved fact there. You can also just ask: *"List all your memories."*
