# Tips & Tricks

Standalone tips for getting more out of Clawpilot — things that surprised you, patterns that work well, and shortcuts worth sharing.

These don't belong to a single use case but are broadly useful.

---

## How to Add a Tip

Add a `.md` file to this folder with a clear, descriptive name (e.g., `be-specific-about-output-format.md`). Keep it short and actionable. Open a PR.
