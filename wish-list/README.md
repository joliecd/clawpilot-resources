# Wish List

Use cases the team *wants* but that don't have a solution yet.

This is the place to document problems worth solving — even if you don't know how to solve them. Others on the team might pick them up, or they may inspire future Clawpilot skill development.

---

## How It Works

1. **Add a wish** — Copy `_template.md`, describe the problem, open a PR
2. **Pick up a wish** — See something you can solve? Build the use case and convert it
3. **Convert to a use case** — Once solved, move to `use-cases/`, link back here
